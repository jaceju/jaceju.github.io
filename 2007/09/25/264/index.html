CSS

  • CSS techniques I use all the time

雖然 * { margin: 0; padding: 0; } 可以消除大部份元素在瀏覽器上顯示的差異，但不幸地它卻也去除掉了一些不錯的預設值。 Montoya 提供了他的經驗，並且也給了一個有別於 reset.css 的參考設定。

這裡我建議一般前端頁面在設計上如果會讓使用者使用線上 HTML 編輯器來建立內容的話，就儘可能不要使用 reset.css 。
