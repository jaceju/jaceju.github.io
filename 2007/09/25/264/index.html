CSS

  • CSS techniques I use all the time

雖然 * { margin: 0; padding: 0; } 可以消除大部份元素在瀏覽器上顯示的差異，但不幸地它卻也去除掉了一些不錯的預設值。 Montoya 提供了他的經驗，並且也給了一個有別於 reset.css 的參考設定。

這裡我建議一般前端頁面在設計上如果會讓使用者使用線上 HTML 編輯器來建立內容的話，就儘可能不要使用 reset.css 。

  • CSS Frameworks + CSS Reset: Design From Scratch

介紹 CSS (排版) 框架還有一些不錯的技巧。

  • IE and the CSS Box Model: Wronger than Wrong

在 IE7 上， DOCTYPE 對 CSS Model 的重要性；其實 IE6 也是一樣。目前比較好的方式就是使用 XHTML 1.0 Transitional 或 XHTML 1.0 Strict 兩種 DOCTYPE ，而我個人推薦用 Transitional 。

  • Almost Bullet-Proof CSS Calendar

把文章的建立日期作成像一個日曆的效果。

  • Developer’s Notebook: Forward-thinking CSS float-clearing

一些 CSS Float 在各家瀏覽器 (特別是 IE) 上 Clear 的技巧，以避掉一些奇奇怪怪的 Bug 。
