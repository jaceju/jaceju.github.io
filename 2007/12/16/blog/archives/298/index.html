jQuery

  • jQuery Live Query Plugin 1.0

通常在利用 AJAX 來更新 element 的 innerHTML 內容後，會讓 element 的 event handler 消失，這時候我們得手動重新 rebind event 到 element 上。 Live Query 這個套件就能自動幫我們處理這件事情，是個很不錯的 plugin 。
