新書預告

以下為最近將會出版的書籍，值得期待：* CSS 大全 第三版

好書推薦
另外整理一些我個人覺得不錯的 Web 開發相關中文書籍供大家參考 (我看過的才介紹) ，它們都是非常值得大家閱讀與收藏的技術書籍。
註：只介紹中文書的原因是為了讓經驗較少的伙伴也能快速瞭解這些技術，事實上還有更多原文的好書值得參考。

CSS 向世界最TOP的網站學 CSS 網頁設計可以學到一些真實的 CSS 設計概念。 Eric Meyer 再談 CSS 網頁排版設計 CSS 大師著作，必讀！

  • The Zen of CSS Design─網頁視覺設計的王道 這本書也可以學到很多 Zen Garden 在 CSS 設計的概念。 網頁設計標準規格 要瞭解標準網頁與 CSS 如何搭配的話，看這本就對了。 CSS 網頁設計師手札 多到不行的 CSS 技巧說明，必讀！

    JavaScript & AJAX JavaScript 網頁設計師手札 用 DOM 的角度來介紹 JavaScript ，值得一讀。 JavaScript 大全

    看完它才能算是瞭解 JavaScript 。中文目前為第 4 版，原文則已經到第 5 版了。不過天瓏網站竟然找不到了，可以找其他家書店看看。 Ajax 網頁程式設計─Google 成功背後的技術 這是台灣第一本有完整實作的 Ajax 書籍。 Ajax 快速上手 維持深入淺出系列的良好風格，個人建議沒碰過 Ajax 的人先看這本。 Ajax Hacks 駭客八十招 介紹很多 Ajax 技術，好書一本！ Ajax 實戰手冊 這本有介紹到實作必須考量的地方，值得參考。* Ajax 設計模式 把 Ajax 相關模式及需要注意的地方解釋得很清楚，必讀！

    PHP PHP 5 徹底研究 PHP 5 有哪些東西？看這本書後就能掌握住七、八成了。 究極 PHP：進階網路應用指南 本書可以學習到很多 PHP 處理網路通訊的技術。 PHP Smarty 樣版引擎 內舉不避親，本書主要透過 Smarty 來介紹團隊開發的方式。 PHP 精華工具集模組、擴充功能、效能加速 本書介紹很多 PHP 工具與技術，值得一看。 專業 PHP 5 程式設計指南 目前我看過最好的 PHP 物件導向中文書籍，可惜買不到了。 PHP 之戀 天瓏竟然還有在賣，本書介紹很多你沒看過的 PHP 應用。* PHP 程式設計專家必備手冊 一大堆的 PHP 技巧，但是也買不到了，還好原文已經有第二版。

    Web Apache 技術手冊 安裝與設定 Apache 必看。 Apache 錦囊妙技 傳授許多 Apache 的秘技，設定 Apache 必看。* Web Site 錦囊妙計 架站必讀好書，不過天瓏不知道為什麼沒有？而且竟然還是上奇出版的歐萊禮書籍…出版界真是令人摸不透呀…

    Design 如何設計好網站 網站不是漂亮就好了，這本書教你如何從使用者的角度去設計網站。 操作介面設計模式 現行的軟體介面有哪些呢？本書整理了許多令人讚嘆的介面設計。

Programming 深入淺出設計模式 設計模式是什麼？這本書看完後包你一清二楚。 物件導向設計模式 設計模式的經典名著，不過我建議先看看「深入淺出設計模式」後再回頭看這本。另外還有精裝本，收藏者必備。 Design Patterns 於 Java 語言上的實習應用 用許多 Java 實例來介紹設計模式，可以補足前面兩本設計模式書籍的實作觀念。 重構 - 改善既有程式的設計 介紹許多簡單的重構手法，可以幫助我們把程式重新修改成較易維護的架構，必讀！ 軟體預先架構之美學 和重構有異曲同工之妙，主要是利用經驗法則來避免掉後續過多的重構動作，值得參考。 極致軟體製程 XP 大師著作，包含許多有趣的軟體開發方法，值得一讀！ 敏捷軟體開發：原則、樣式及實務 想走物件導向軟體開發必讀書籍！ 深入淺出物件導向分析與設計如果覺得市面上的 OO 分析書籍太過於艱深的話，這本入門書絕對值得一讀！

當然一定還有其他中文的好書，只是我力有未逮無法一一拜讀；這裡也歡迎大家一起推薦自己心目中的好書，讓彼此的功力進步再進步。
