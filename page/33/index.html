[JavaScript] jQuery 1.2.1 釋出

觀望得果然沒錯， jQuery 1.2.1 釋出了。

jQuery 1.2.1: Quick Fixes for 1.2

1.2.1 版解決了一些 IE 上 Ajax 的問題， 然後也把 eq() 這個方法又放了回來，免得有一堆 plugins 不能用。

話說阿土伯好像很忙，還沒看到他對 jQuery 的新版本表示一下意見。
