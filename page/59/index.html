連結 CSS 樣式說明

發表於 2005-04-08   |  

將元素的 position 指定為 relative 後，會使得元素的寬高受到父元素寬高的影響。 例如 a 標籤被包在 li 標籤內時，如果 li 標籤寬度設定為 100px ， a 標籤的 Box 寬度 (等於 margin + border + padding + width) 就會等於 100px。這樣一來，連結的作用區域就不會只受限於文字或圖片上，而可以佔滿整個父元素。
