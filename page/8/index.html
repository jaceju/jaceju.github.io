發表於 2011-07-03   |

LiveReload 是 Ruby 的一個 gem 套件，使用時必須先安裝好 Ruby 的開發環境。比較常見的 Ruby 開發環境是以 Apple Mac 系統為主，因此網路上大部份的教學也都是用 Mac 系統做示範；但是很多公司目前還是以 Windows 為主要的開發環境，所以本文就簡單為大家介紹如何在 Windows 上使用 LiveReload 。
