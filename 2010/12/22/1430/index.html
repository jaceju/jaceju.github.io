Web 開發之路上的大哉問

  • 心得感想
星期二, 二月 2日 2016

分享幾個我心中一些有關 Web 開發的疑問，歡迎大家一起討論。當然這些問題沒有固定的標準答案，希望大家能分享一下自己的經驗。

更新：將會在我的 Plurk 上以一天一則的方式來討論。

語言與開發

  • 不論是前端或後端，為什麼 Query 語法漸漸成為一種趨勢？

  • 事件驅動在函式導向和物件導向中的實作有什麼不同？

  • 設計模式對 Web 開發的意義是什麼？

  • 測試驅動開發有什麼優缺點？

資料庫

  • 你會選擇將商業邏輯放在程式端 (例如 PHP ) 還是在資料庫端 (例如 Stored Procedure ) ？

  • 怎麼測試需要存取資料庫的程式？

  • Key-Value Database 的優勢是否值得現在就用？

手持裝置

  • 手持裝置的逐漸普遍，將會為 Web 開發帶來什麼樣的變化？

  • 現在學習 HTML5 及 CSS3 是否能為手持裝置的開發加分？

瀏覽器

  • 在客戶指明 IE 6/7 是不可放棄的瀏覽環境時，該怎麼應對？

  • 你是否會在 Web 應用程式中使用 Unobtrusive JavaScript 技術？

系統

  • 在面臨效率瓶頸時，你會選擇修正程式碼還是增加機器？
