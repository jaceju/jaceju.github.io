在宗董的 Blog 看到這篇：[交換兩個變數 (不使用 tmp 變數) 程式寫法](http://plog.longwin.com.tw/programming/2007/11/23/variable_swap_programming_2007)，本來想留言，不過宗董的 Blog 系統似乎有問題。

宗董的方法是這樣的：

```
$a ^= $b;
$b ^= $a;
$a ^= $b;

```

我是想說既然是用 PHP 了，就應該好好善用一下 PHP 的原生語法：

```
list($a, $b) = array($b, $a);

```

搞定~~
