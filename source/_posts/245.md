原來我一直錯怪樂多...

之前的搜尋表單值是傳送到：

```
http://cgi.blog.roodo.com/search

```

因為 mark 和 kuni 說搜尋有錯誤，所以我剛剛又到樂多 Blog 首頁去測了一下，發現正確的位址應該是：

```
http://search.blog.roodo.com/index.php

```

特此更正一下。

現在搜尋終於正常了，為我之前的錯誤向各位道歉。
